Список изменений в обновлении 1.31


Дополнения и изменения:

+ Теперь вам ненужно делать логин каждый раз при старте программы. Все ваши персонажи будут автоматически залогинены.
+ Добавлена возможность убирать программу в трей. При зменении системы с залогиненным персонажем в трее будет выведено сообщение со статиками и номером системы.

Исправленные ошибки:

+ Вылет программы при вводе во вкладке "Solar System" названия солнечной системы в неверном регистре.

Patch notes for 1.31 Release 

Features & Changes:

+ Now you have to do unnecessary login every time you start the program. All your characters will be automatically logged in.
+ Added the ability to clean up the program to tray. When pilot change solar system location tray will display a message with the static and the system number.

Defect Fixes:

+ Fixed a bug when entering the program in the tab "Solar System" the solar system name in the wrong case.
